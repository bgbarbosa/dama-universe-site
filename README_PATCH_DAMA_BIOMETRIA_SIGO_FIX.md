# Patch — Dama Biometria SIGO Fix

Este pacote adiciona o programa **Dama Biometria SIGO Fix** ao Dama Universe sem alterar a página do Dama Cleaner.

## Arquivos incluídos

```text
src/data/programs.ts
src/data/updates.ts
src/app/programas/dama-biometria-sigo-fix/page.tsx
public/images/programs/dama-biometria-sigo-fix.png
public/images/programs/dama-biometria-sigo-fix.ico
```

## Link direto de download

```text
https://drive.usercontent.google.com/download?id=1y0pRu7YeazLrSYFq2Sf0rgtrgMkh48rh&export=download
```

## Após copiar os arquivos

Rode:

```powershell
npm run build
```

Se passar:

```powershell
git add .
git commit -m "Adiciona Dama Biometria SIGO Fix ao site"
git push
```

A Vercel deve publicar automaticamente.

## Observação importante

A página contém bloco destacado com a declaração de independência:

- não é ferramenta oficial do SIGO;
- não é produto institucional do SIGO;
- não substitui suporte oficial, equipe de TI, mantenedores do sistema ou contrato de manutenção;
- é uma ferramenta independente para apoio em problemas locais recorrentes de Java, biometria, assinatura e componentes relacionados ao uso do SIGO Desktop.
